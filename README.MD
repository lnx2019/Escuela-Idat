# Sistema de Gestión de Citas Médicas API

## Descripción del Proyecto

Este proyecto consiste en el desarrollo de una **API RESTful para la gestión de citas médicas**, implementada utilizando **ASP.NET Core (.NET 8)**.

El sistema permite administrar:

- Pacientes
- Médicos
- Citas médicas

La API permite realizar operaciones CRUD (Crear, Leer, Actualizar y Eliminar) sobre cada una de estas entidades mediante endpoints HTTP.

El objetivo del proyecto es demostrar la implementación de una **API REST estructurada**, utilizando **controladores, modelos, validaciones y middleware de manejo de errores**.

---

# Tecnologías Utilizadas

- .NET 8
- ASP.NET Core Web API
- C#
- Swagger / OpenAPI
- Postman
- JSON
